## 🔑 关键发现

### 技术采用情况

1. **单细胞转录组 (GEX) 为核心** - 47.2% 的样本包含 GEX 数据
   - 多数使用 10X Genomics Chromium 平台 (GPL24676)
   - 库策略: RNA-Seq 或 OTHER

2. **蛋白质表型检测 (ADT/CITE-seq) 最常见** - 26.8% 样本有蛋白标记
   - 通常与 GEX 配套使用
   - 使用 Antibody-Derived Tags (ADT)

3. **免疫受体库分析 (TCR/BCR) 普遍** - 20.7% 样本含VDJ信息
   - TCR: 117 个样本 (T细胞受体库)
   - BCR: 14 个样本 (B细胞受体库)

4. **多重化样本标记 (HTO)** - 2.1% 样本使用 Cell Hashing
   - 允许在单个测序运行中合并多个样本
   - 节省成本并减少批次效应

5. **ATAC-seq 少数应用** - 仅 3.2% 样本
   - 仅在 GSE305370 中使用
   - 与 GEX + ADT 配套

### 平台分布

| 平台 | GPL ID | 使用频率 | 特点 |
|------|--------|---------|------|
| 10X Genomics | GPL24676 | 主流 | 3' mRNA + CITE-seq |
| 10X Genomics (5') | GPL34281 | 部分 | 5' mRNA + VDJ |
| SMART-seq | GPL16791 | 部分 | 全转录长读 |
| 其他 | GPL30173等 | 少见 | 特殊应用 |

## 📊 样本来源与生物学背景

### 主要组织类型
- **血液样本 (PBMCs)**: 最常见，大多数数据集
- **肿瘤组织**: 部分CRC (结直肠癌) 研究
- **器官组织**: 肝脏、结肠等
- **免疫细胞**: 单独富集的T细胞、B细胞、嗜碱性粒细胞

### 研究主题
- HIV 感染与免疫反应
- 癌症免疫学 (CRC, 肝转移)
- 蛋白质组分析与单细胞异质性
- 细胞表面标记与转录组关联
- 神经退行性疾病 (阿尔茨海默病)
